# Синтетический признак

Ноутбук, демонстрирующий создание синтетических признаков для выявления взаимодействий между переменными в моделях машинного обучения. В работе рассматривается подход feature engineering, при котором новые признаки формируются на основе комбинаций существующих категориальных переменных. Такой метод позволяет моделям лучше улавливать скрытые зависимости между признаками и повышать качество предсказаний. В ноутбуке показан процесс генерации синтетических признаков, их добавление в обучающую выборку, а также проведено сравнение моделей с исходными и расширенными признаками. Дополнительно выполняется анализ полученных результатов, позволяющий оценить влияние синтетических признаков на качество модели и сделать выводы об эффективности данного подхода.
